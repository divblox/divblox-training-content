Functions
  • Function 1: This function will generate a bunch of categories and accounts. It will then generate a bunch of tickets, each linked to a random category and account with a random status and due date.
  • Function 2: Return all tickets in the category specified in the input box
  • Function 3: Return all tickets where the account's first name is specified in the input box and the ticket's status is In Progress
  • Function 4: Return all tickets that have a status Completed where the due date is in the current month, ordered by Ticket Due Date ascending
  • Function 5: Return a list of account full names with a count of tickets that they each currently have In Progress.
  • Function 6: Return a list of account email addresses. For each account, show an array of categories. For each category in the array, show the total count for all tickets in that category, as well as the count for the specific account.
  • Function 7: Optimise function 6 to improve the response time from the server. This is going to be evaluated using the 'Network' tab in your browser's dev tools.
